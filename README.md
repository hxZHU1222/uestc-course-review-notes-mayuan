# UESTC Course Review Notes: 马克思主义基本原理

<img width="2482" height="1236" alt="image" src="https://github.com/user-attachments/assets/c3dff145-fa5b-4389-89cf-d87152d0118b" />


这是一个用于《马克思主义基本原理》课程期末复习的纯前端网页应用，面向个人复习、知识点背诵、简答题整理、选择题练习和思维导图浏览。当前版本已更新为浅色 Apple 风格界面，并把“今日复习”、背诵检验、题目状态标记和导图交互整合在同一个入口中。

## 启动入口

在线启动链接：[https://hxzhu1222.github.io/uestc-course-review-notes-mayuan/](https://hxzhu1222.github.io/uestc-course-review-notes-mayuan/)

默认访问密码：

```text
mayuan
```

> 注意：网页会通过 `fetch` 加载 `data/` 目录下的 JSON 数据。在线版本可直接打开；本地预览时请使用静态服务器启动，不建议直接双击 `index.html`。

## 本地运行

在项目根目录运行：

```bash
python -m http.server 8000
```

然后打开本地启动链接：[http://127.0.0.1:8000/](http://127.0.0.1:8000/)

如果 `8000` 端口被占用，可以换成其他端口，例如：

```bash
python -m http.server 8080
```

对应启动链接为：[http://127.0.0.1:8080/](http://127.0.0.1:8080/)

## 功能模块

本项目目前包含以下复习模块：

* **今日复习**：集中展示掌握进度、下一轮复习入口和高频操作。
* **背诵卡片**：包含 215 张讲义背诵卡，支持连续、网格、单张和检验模式。
* **背诵检验**：通过遮罩挖空重点词句，点击后显示答案，适合考前自测。
* **卡片列表**：集中浏览全部知识点卡片，并支持按章节、Day、状态等条件筛选。
* **简答题复习**：包含 61 道简答题，保留默写区和关联知识点。
* **选择题练习**：包含 400 道选择题，支持查看答案、关联知识点和做题状态标记。
* **思维导图浏览**：包含 13 页导图和 236 个节点，支持页码切换、搜索、缩放、全屏和小地图定位。

复习进度、卡片状态和题目状态会保存在浏览器 `localStorage` 中。

## 使用建议

1. 先通过“今日复习”确认当前掌握进度。
2. 使用“背诵”模块快速过一遍核心知识点。
3. 切换到“检验”模式，对重点句子进行遮罩式自测。
4. 通过“简答题”模块整理主观题答题框架。
5. 使用“选择题”模块查漏补缺，并标记做题状态。
6. 用“思维导图”模块建立章节结构感，遇到薄弱节点再回到关联卡片复习。
