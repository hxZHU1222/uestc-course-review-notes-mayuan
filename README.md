# UESTC Course Review Notes: 马克思主义基本原理

这是一个用于《马克思主义基本原理》课程期末复习的静态网页项目，主要面向个人复习、知识点梳理、背诵卡片、选择题练习、简答题复习和思维导图浏览。

项目采用纯前端静态页面实现，可以直接通过浏览器打开，也部署到 GitHub Pages了。


## 项目内容

本项目目前包含以下复习模块：

* **讲义背诵卡片**：将课程讲义内容整理为适合背诵的知识卡片。
* **卡片列表**：集中查看全部知识点卡片，便于快速定位。
* **简答题复习**：整理常见简答题，适合考前集中复习。
* **选择题练习**：整理选择题题库，并支持查看答案与知识点关联。
* **思维导图浏览**：将课程思维导图拆分为网页可浏览图片。
* **资料/检查页面**：展示数据抽取结果、覆盖情况与解析检查信息。

## 使用建议

本项目适合以下复习方式：

1. 先通过“背诵”模块快速过一遍核心知识点。
2. 再通过“简答题”模块整理主观题答题框架。
3. 使用“选择题”模块查漏补缺。
4. 用“思维导图”模块建立章节结构感。
5. 在考前重点查看未掌握、未背熟或容易混淆的内容。
